Math Application

This Math application was created by using Django, Bootstrap, CSS, HTML, JavaScript, Node.js, Express, Socket.io, and MongoDB. This application teaches students from all college levels, starting from Algebra all the way through Differential Equations. In the application is added a forum section and a schedule class with me for any questions they have in mind. Any person is allowed to create an account and have a wonderful journey in the math learning process.

Users have the opportunity to create a new account by using their username and password.
